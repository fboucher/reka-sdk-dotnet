# Reka SDK for .NET

Official .NET SDK for integrating Reka AI APIs into your .NET applications.

## Installation

Install the package via NuGet:

```bash
dotnet add package Reka.SDK
```

Or via Package Manager Console:

```powershell
Install-Package Reka.SDK
```

## Features

- Easy-to-use .NET interface for Reka AI APIs
- Support for API: chat and research
- Strongly-typed responses and request models


## Quick Start

```csharp
using Reka.SDK;

// Your code here to use the Reka SDK
```

## Requirements

- .NET 9.0 or later
